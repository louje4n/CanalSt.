#CanalSt.
Aesthetic Intelligence & Archival Alternatives.

High-end streetwear is locked behind absurd resale markups. CanalSt. is an underground, brutalist P2P marketplace built to change that. We promote near lookalikes and high-quality dupes, giving everyone access to that luxury feel without the financial gatekeeping.

I built this platform to treat style alternatives with the exact same editorial respect as the original pieces they reference. Backed by rigorous engineering, CanalSt. replaces thrift store chaos with a high-velocity, trusted trading floor for aesthetic alignment.

Core Features
Automated Curation: Sellers upload a photo. The system instantly generates a technical specification table, identifies the aesthetic lineage, and estimates market demand.

Magazine Rhythm Feed: The "Archive" feed abandons the standard infinite scroll. It uses a dynamic rhythm of full-bleed Heroes, high-density Rotation rows, and 2-column Grids to keep the experience visually engaging.

High-Fidelity Market Signals: Real-time data density is crucial for buyer confidence. Every listing features live watcher counts, pulsing high-demand indicators, and 90-day price drop micro-graphs.

Separation of Trust: The platform structurally separates the Verified Seller rating from the Authenticated Item badge, solving the biggest anxiety in the alternative market.

Technical Architecture
Built with an obsession for performance, minimalism, and strict data security.

Frontend: Swift 6 and SwiftUI. Engineered with a strict MVVM-C pattern, prioritizing 60fps scrolling through high-res imagery and a brutalist, monochromatic UI (pure blacks, stark whites, monospaced typography).

Backend: Supabase (PostgreSQL).

Security: Deeply integrated Row-Level Security (RLS) policies to protect P2P transaction integrity and user data. Authentication flags are strict server-side operations to prevent client-side manipulation.
