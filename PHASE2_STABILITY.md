# Phase 2: Architecture Stability & Modernization

## 1. Objective
Upgrade the CanalSt. React prototype into a production-ready, stable architecture. We are migrating to React Router for navigation, Zustand for global state management, and TypeScript for type safety.
**Constraint:** The brutalist aesthetic, exact hex colors, and "Studio Plate" components must remain visually identical. Do not alter the UI design.

## 2. Step 1: Dependency Installation
Run the following commands to install the required architectural packages:
```bash
npm install react-router-dom zustand
npm install -D typescript @types/react @types/react-dom
